# Code de Andas Insulation

Bienvenido al repositorio de Code de Andas Insulation.

## 🎯 ¿Qué es este proyecto?

Este es el proyecto que contiene todo el código y los archivos necesarios para crear el sitio web de **Andas Insulation**. Piensa en él como una carpeta maestra donde guardamos todas las piezas que forman nuestro sitio web.

---

## 📁 Estructura del Proyecto

La carpeta principal se divide en varias subcarpetas, cada una con un propósito específico:

### **imagenes/** 
Aquí guardamos todas las **imágenes** que usamos en el sitio web. Son archivos visuales que ayudan a que el sitio sea más atractivo y fácil de entender.

### **js/** 
**JavaScript** es un lenguaje de programación que hace que el sitio web sea **interactivo**. Los archivos en esta carpeta contienen el código que hace funcionar cosas como:
- Botones que responden cuando haces clic
- Menús que se despliegan
- Animaciones y efectos especiales

### **videos/** 
Aquí almacenamos archivos de **vídeo** que queremos mostrar en el sitio web. Por ejemplo, videos de demostración o testimonios.

### **fotos/** 
Esta carpeta es para guardar **fotografías** específicas del proyecto. Pueden ser fotos de productos, equipo de trabajo, o galerías especiales.

### **css/** 
**CSS** es el lenguaje que controla cómo se **ve** el sitio web:
- Los colores de los botones
- El tamaño de los textos
- Las posiciones de los elementos
- Los fondos y bordes

Sin CSS, el sitio sería solo texto plano. Con CSS, se ve profesional y bonito.

---

## 🔧 ¿Cómo funciona todo junto?

1. **HTML** (la estructura) = Como los huesos de tu cuerpo
2. **CSS** (los estilos) = Como la ropa y maquillaje que te hace verse bien
3. **JavaScript** (la interactividad) = Como los músculos que te permiten moverte
4. **Imágenes, videos y fotos** = Como los accesorios que complementan tu look

Cuando alguien visita el sitio web, ve todo esto junto funcionando como una máquina perfecta.
